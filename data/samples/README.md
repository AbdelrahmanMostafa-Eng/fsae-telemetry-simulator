# Sample Data

This folder contains small, synthetic datasets used for:
- Demonstrations
- Tutorials
- Testing
- Documentation examples
- Quick experiments without requiring real telemetry or track files

These datasets are intentionally lightweight and easy to understand.

## Purpose

Sample data allows you to:
- Run example scripts immediately
- Test analysis and visualization modules
- Validate data‑loading utilities
- Provide reproducible examples in documentation

## Notes
- These files are **not** meant to represent real data.
- They should remain small and easy to parse.
- Real telemetry, real tracks, and real setups belong in their respective folders.
